# digital-clock-display

時間と集中のためのフルスクリーン・クロック＆タイマー。

## Overview

視界に入る情報を絞り、作業のリズムを作る。時計・ポモドーロ・タイマーを一枚に集約。

## Screens

### Web

ブラウザで即起動。視認性重視の表示と柔軟なカスタマイズ。

## Core

- Clock：`HH:mm:ss` / `HH:mm`、日付表示の切替。
- Pomodoro：作業・休憩時間とセット数を調整、自動進行、進捗リングと次の状態を表示。
- Timer：時・分・秒で設定、円形プログレスで残り時間を可視化。

## Experience

- 画面移動はタブ／スワイプ／左右キー。
- Picture-in-Pictureとフルスクリーンに対応。
- カーソルと操作UIの自動非表示。
- ポモドーロ／タイマーの操作UIの表示切替。

## Visual

- ライト／ダークの切替。
- 文字色のカスタム。
- すりガラス効果の切替。
- パネル背景の表示切替。
- 背景タイプは画像／白／黒／透明／単色。

## Background

- Unsplashカテゴリから画像取得。
- 一定間隔で自動更新。
- ローカル画像のアップロードに対応。

## Install

- PWAとしてホーム画面に追加。

## Quick Start

1. 上部タブで「時計 / ポモドーロ / タイマー」を切り替える。
1. 右下の設定ボタンで背景・時計・タイマーの表示を調整する。
1. Picture-in-Picture とフルスクリーンを右下アクションから切り替える。
1. スマホは左右スワイプで画面を移動する。
